# 🌐 Android Network Tools

Network diagnostics and testing tools for Android.

## Tools

- `dns_check.py` — Test DNS resolution and detect hijacking
- `latency_monitor.py` — Real-time ICMP latency to multiple targets
- `arp_spoof_detect.sh` — Detect ARP spoofing on local network
- `network_speed.py` — Quick speed test via ADB
- `captive_portal_check.py` — Detect if on a captive portal
